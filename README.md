# Doodle Chat

A chat frontend for the Doodle frontend challenge, goal is to send and display messages
from all senders.

## Stack

- **Vite + React + TypeScript**
- **TanStack Query**
- **Tailwind CSS**

## Setup

The [chat API](https://github.com/DoodleScheduling/frontend-challenge-chat-api)
must be running locally first.

```sh
cp .env.example .env   # API base URL + bearer token
npm install
npm run dev
```
